# ИКТ Поддршка во Модни Дизајни
Предмет: ИКТ за развој  
Студент: Хирије Заими

#Опис
Проектот се фокусира на дигитализација на традиционалното модно занаетчиство и поддршка на модни дизајни преку ИКТ алатки и SDG цели.

#Методологија
Користен е животен циклус на развој на софтвер (SDLC) организиран во фази:
1. Анализа
2. Дигитализација
3. Лансирање
4. Евалуација
